Perguntas Frequentes

Entenda como a plataforma The APEX (Core) funciona.

Como funciona a análise na versão Core?
A análise na versão Core foca em transparência e processamento local:
  • Análise Estática Local: Verificamos arquivos PE/ELF localmente buscando strings suspeitas, importações maliciosas e anomalias de estrutura sem enviar o arquivo completo para a nuvem.
  • Reputação Externa: Consultamos o VirusTotal (via API) para obter vereditos de múltiplos motores antivírus.
  • OpenSourceMalware: Utilizamos inteligência aberta para identificar assinaturas conhecidas de malware.
  • Detecção de Phishing: Analisamos URLs em busca de ataques de homógrafo e conteúdo sensível.
Por que não vejo opções de IA ou SIEM?
O The APEX segue um modelo Open Core. As funcionalidades de Inteligência Artificial (IA Cognitiva), Integração Nativa com SIEM (Elastic/Wazuh) e Network Sniffer de nível de kernel são exclusivas dos Tiers Operador e Enterprise.

A versão Core (Open Source) fornece a fundação tecnológica e os motores de análise estática essenciais para pesquisadores e auditores.
O que é a Detecção de Phishing Avançada?
O sistema identifica URLs que utilizam caracteres especiais (como cirílicos) para se passar por domínios legítimos. Essa técnica, conhecida como ataque de homógrafo, é comum em tentativas de phishing e é detectada automaticamente pela nossa ferramenta de análise de URL.
Guia: Como obter as API Keys permitidas no Core

A versão Core permite integração com dois serviços principais:

1. VirusTotal (Análise de Reputação)
  • Cadastre-se em VirusTotal Join Us.
  • Após logar, clique no seu perfil (canto superior direito) e vá em "API Key".
  • Copie a chave exibida em "Personal API Key".
2. OpenSourceMalware (Inteligência Aberta)
  • Acesse OpenSourceMalware.com para mais informações sobre o projeto.
  • A chave de API pode ser configurada diretamente se você possuir acesso ao serviço de inteligência deles.
Onde configurar as chaves?
Após obter suas chaves, clique no ícone de engrenagem no menu lateral ou acesse a página de Configurações. Cole as chaves nos campos correspondentes (VirusTotal ou OpenSourceMalware) e clique em "Salvar Alterações".
